Project

Mp3Player

A Java & JavaFX desktop music player with full playlist management.

  • Java
  • JavaFX
  • OOP

Overview

Mp3Player is a desktop music player built entirely in Java and JavaFX. It lets users browse and load songs from their device, add them to a custom playlist, and manage playback directly within a clean, easy-to-use interface, no external dependencies, just core Java and object-oriented design.

Key Features

  • Play, pause, and restart the current track
  • Skip forward and back through the playlist
  • Adjustable playback speed control
  • Editable song list , add or delete tracks on the fly
  • Live progress bar with seek support
  • Built entirely with core Java and JavaFX, following object-oriented design principles
