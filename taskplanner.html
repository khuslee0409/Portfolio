Task Planner for Students

Task Planner is a full-stack productivity application designed to help students manage tasks, deadlines, and academic workload efficiently. The application provides secure authentication, persistent task storage, and a clean desktop interface built with modern Java technologies. This project demonstrates my ability to design, build, deploy, and maintain a complete production-ready system.

Key Features

  • Secure user registration and login with JWT authentication
  • Email verification during account creation
  • Create, update, delete, and manage tasks
  • Persistent database storage (MySQL)
  • RESTful API architecture
  • Desktop frontend built with JavaFX
  • Deployed backend on cloud hosting (Railway)
